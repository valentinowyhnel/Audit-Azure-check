# 📋 Checklists d'Audit

## 🏗️ Architecture AD
- [ ] **AD-ARCH-01** : Inventorier tous les Domaines et Forêts AD.
- [ ] **AD-ARCH-02** : Vérifier les rôles FSMO.
- [ ] **AD-ARCH-03** : Contrôler la configuration des DCs (OS, Patchs).

## ☁️ Azure / Entra ID
- [ ] **AZ-ID-01** : Vérifier l'activation obligatoire du MFA pour tous les administrateurs.
- [ ] **AZ-ID-02** : Analyser les politiques de Conditional Access.
- [ ] **AZ-ID-03** : Inventorier les Applications à haut privilèges (Microsoft Graph API).

## 🔗 Hybrid Identity
- [ ] **HYB-SYNC-01** : Vérifier les permissions du compte de synchronisation AD Connect.
- [ ] **HYB-SYNC-02** : Analyser la configuration du Pass-through Authentication (PTA).
- [ ] **HYB-SYNC-03** : Isoler physiquement/logiquement le serveur AD Connect.
