# 🛠️ Outils d'Audit Azure sur GitHub

Le tableau ci-dessous présente les outils les plus complets et spécialisés pour vos audits.

| Nom de l'outil | Lien GitHub | Description | Popularité |
| :--- | :--- | :--- | :--- |
| **Prowler** | [github.com/.../prowler](https://github.com/prowler-cloud/prowler) | Outil multi-cloud (CIS, NIST). Idéal pour la conformité automatisée. | 🔥 Très élevée |

## 🔑 Outils Clés AD/Azure AD (Red Team/Pentest)

- **BloodHound CE** : Analyse graphique des chemins d'attaque.
- **SharpHound / AzureHound** : Collecteurs de données pour BloodHound.
- **AADInternals** : Administration et vulnérabilisation Azure AD/M365.
- **PingCastle** : Analyseur de santé et de sécurité AD.
- **Rubeus** : Attaques Kerberos (Kerberoasting, AS-REP Roasting).
